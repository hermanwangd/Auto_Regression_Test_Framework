Evidence records shall preserve acceptance-criteria source refs, optional Product/RP labels when generated, test case ID, batch ID, run ID, source artifact links, approval records, and waiver/manual-only exclusion records where applicable. Framework verification reports may use sample fixture identifiers, but they shall not be represented as downstream Product/RP release evidence. Current-stage framework maturity uses EVD-000, EVD-009, EVD-010, and EVD-011 first; EVD-001 through EVD-006 are next-stage Phase 2 Agent Skill support evidence.
