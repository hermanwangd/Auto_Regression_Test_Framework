# Support

## Provider Support Policy

Public provider support is expressed by `support_status` in [provider_support_matrix.md](docs/09-operations/provider_support_matrix.md).

- `supported`: provider contract and runtime path are release-verified.
- `contract_only`: public contract exists, but runtime execution is blocked or not release-verified.

Server ownership terms such as framework-managed, external, native, mock, or ephemeral describe Env_Profile/runtime coverage. They are not public provider support statuses.

## Questions and Issues

Open an issue with framework version, command, suite path, profile, result JSON path, evidence path, and redacted logs. Do not include raw secrets.
